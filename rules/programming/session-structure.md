# Session Structure

## Summary

Rules governing session timing, set structures, series composition, rest periods, and warm-up sets. Every session is 40 minutes including warm-up.

## Rules

### Session Timing

- Every session is **40 minutes** including warm-up.
- Programs should not exceed **35 minutes** of work time.
- Typical time splits:
  - Option 1: 15 min A-series, 10 min B-series, 10 min C-series.
  - Option 2: 20 min A-series, 15 min B-series.

### Set Structures

- **Standard:** 3-set A-series, 3-set B-series, 2-set C-series.
- **For quicker clients:** 4-set A-series, 2-set B-series, 2-set C-series.
- **Other options:** 4–5 set A-series and 3-set B-series only; B capped at 3 sets max.
- If program is fully bilateral, some clients can handle 3-3-2 or 4-2-2 sets (A-B-C).
- If there is single-limb work, cap at 3-2-2 sets for A, B, C.

### Series Composition

- **Maximum two exercises per series.** No trisets.
  - Correct: A-Series = Press + Row.
  - Incorrect: A-Series = Press + Row + Sit Up.
- No pairing of single-limb upper with double-limb upper (same rule for lower).
- Single-limb upper can be paired with double-limb lower and vice versa.
- Only one single-limb series per day.

### Rest Times

- Under 6 reps: **120 seconds** (180 possible but must fit within session).
- 7–12 reps: **90 seconds**.
- 12+ reps: **75 seconds**.
- Coaches' discretion, but rest should be 60–120 seconds where appropriate.

### Warm-Up Sets

- Sparing in B and C-series.
- Only add for advanced clients with heavy lifts.

## Site Exceptions

> No site-specific exceptions currently.

## Examples

**Example 1: Standard 3D session**

A-Series (3 sets, 15 min): Barbell Squat + Cable Row. B-Series (3 sets, 10 min): DB Press + Leg Curl. C-Series (2 sets, 10 min): Bicep Curl + Plank.

**Example 2: Single-limb day**

Because a single-limb exercise is included, sets are capped at 3-2-2 (A-B-C).
